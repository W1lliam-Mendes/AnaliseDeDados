# Projeto de Análise de Dados

Ao analisar o faturamento desta rede de lojas, descobri que cerca de **75%** do resultado total vem de uma única unidade: o **Iguatemi Campinas**.

Ao investigar o porquê, notei que o produto **'Bermuda Liso'** é o grande responsável por esse desempenho, movimentando **R$ 36.581** apenas nesta loja.

## Proposta de Melhoria

- Analisar o perfil de consumo e a estratégia de estoque da unidade que mais vende.
- Propor a redistribuição do produto **'Bermuda Liso'** para as outras lojas.
